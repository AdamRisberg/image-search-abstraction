Image Search Abstraction

Performs an image search and returns ten results at a time with pagination. Can also view latest searches performed using the api.

Image Search Examples

  • Search for "cats": https://fast-ocean-74431.herokuapp.com/api/imagesearch/cats
  • Search for "cars", second page of results: https://fast-ocean-74431.herokuapp.com/api/imagesearch/cars?offset=2
